# Mothra

A FreeBSD send-pr command line tool for bugzilla system.

[https://bugs.freebsd.org/bugzilla/](https://bugs.freebsd.org/bugzilla/)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mothra'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mothra
